# Translation Consistency Checklist

When updating `en/` content, keep `ua/` and `other/` aligned:

- [ ] Headings match across all three languages (same structure, translated wording)
- [ ] Internal links use the same folder names (e.g., `../01-cloud-adoption/`, `../90-roadmap/`)
- [ ] Image file names are identical across language folders
- [ ] New sections added to `en/` are either reflected or marked `<!-- TODO: translate -->` in `ua/` and `other/`
- [ ] Exercise numbers and module structure stay consistent across translations
